<task type="auto">
  <name>Task 2: Add smooth color transitions for theme switching</name>
  <files>frontend/project/app/globals.css</files>
  <action>
Add CSS transitions so theme changes fade smoothly (~250ms) rather than snapping instantly.

In the `@layer base` section (the one with `* { border-color: ... }`), update to:

```css
@layer base {
  * {
    border-color: hsl(var(--border));
    transition-property: color, background-color, border-color;
    transition-duration: 250ms;
    transition-timing-function: ease-in-out;
  }
  body {
    background-color: hsl(var(--background));
    color: hsl(var(--foreground));
  }
}
```

Also add a reduced-motion media query for accessibility:

```css
@media (prefers-reduced-motion: reduce) {
  * {
    transition-duration: 0ms !important;
  }
}
```

Do NOT add `transition: all` - only transition color-related properties to avoid performance issues (per RESEARCH.md pitfall #5).
  </action>

Run `grep "transition-property: color" frontend/project/app/globals.css` to confirm transitions added.
Run `grep "prefers-reduced-motion" frontend/project/app/globals.css` to confirm accessibility query.
  </verify>
  <done>
Theme transitions configured for 250ms fade on color, background-color, and border-color. Reduced motion respected.
